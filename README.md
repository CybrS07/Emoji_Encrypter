## 🚀 Overview

**Emoji Encrypter** is a Python-based tool designed to map standard ASCII characters to a unique set of emojis. While it isn't meant for high-stakes cryptographic security, it provides a fun, visual way to hide messages in plain sight.

### 🎨 Visual Transformation
*   **Input:** `Hello World!`
*   **Encrypted:** `🔪👻🔥🔥🛸⬛🌘🛸⚖️🔥👺⚠️`

---

## 🛠️ Features

*   **Bidirectional:** Encrypt text into emojis and decrypt it back to the original message.
*   **Customizable Mapping:** Easily modify the `map.py` (or dictionary) to change which characters correspond to which emojis.
*   **Zero Dependencies:** Runs on standard Python 3.x with no need for `pip install`.
*   **Cross-Platform:** Works wherever Python runs—Linux, macOS, or Windows.

## 💻 Quick Start

### 1. Clone the Repository
```bash
git clone [https://github.com/CybrS07/Emoji_Encrypter](https://github.com/CybrS07/Emoji_Encrypter)
cd Emoji_Encrypter
```

### 2. Execution
Go to the folder and run the main.py file
```bash
python emoji_encrypter.py
```
